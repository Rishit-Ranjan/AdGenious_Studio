## AdGenius Studio

How to run:
Clone the repository.
run npm install
install the requirements from requirements.txt for backen in the .venv
run the backend: uvicorn main:app --reload --port 8000 in the .venv environment
Then, run the frontend: npm run dev<br/><br/>
